Diet Recommendation System
A smart Diet Recommendation System that suggests personalized diet plans based on user inputs such as age, gender, BMI, health conditions, and lifestyle preferences.
The goal of this project is to promote healthy eating habits using data-driven recommendations.

Problem Statement
Many people struggle to maintain a balanced diet due to lack of personalized guidance. Generic diet plans often fail to consider individual health conditions and nutritional needs.
This system aims to recommend customized diet plans to improve overall health and well-being.

Objectives
Provide personalized diet recommendations
Consider user health parameters and preferences
Promote balanced nutrition
Assist users in making healthier food choices

Approach
The system works by:
Collecting user inputs (age, weight, height, BMI, health conditions, diet type)
Processing data using predefined nutritional rules / ML model
Matching user profiles with suitable diet plans
Displaying recommended meals and nutrition tips

Technologies Used
Python
Pandas & NumPy – Data handling
Scikit-learn – Machine Learning (if applicable)
Flask – Web framework
HTML/CSS – Frontend
Jupyter Notebook – Model training & analysis

Dataset Description
The dataset includes:
Age
Gender
BMI
Health conditions (Diabetes, BP, etc.)
Food preferences (Veg/Non-Veg)
Recommended meals
